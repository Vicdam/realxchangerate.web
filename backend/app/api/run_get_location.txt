How to make sure get_location_currency runs fine

1. Install dependencies
	```pip install -r requirements.txt```
2. Install chromedriver and add to path
	* unzip chromedriver_linux64.zip file
	* Move chromedriver to a location in path by running the following command
		* Check directories in path
			```echo $PATH```
		* move driver to one of them
			e.g
			```mv chromedriver /usr/local/bin/```

You are now ready to run file
